Instellingen menu

Vervolgkeuzes

  • Taal instellen
    (Nog) niet functioneel, mogelijkheid om de taal van GUI intestellen.
  • Loglevel
    Voor debugging doeleinden, gedetailleerde messages kan worden ingesteld (standaard is het detail niveau INFO).
  • Uiterlijk
    Het uiterlijk van de Windows kan worden gewijzigd.
  • Creeer logbestanden
    Verwijst naar de directory waar de log-bestanden worden gecreeerd.
  • Dubbele kaartoverzichten in Tab
    Default is False, er worden geen meerdere kaarten toegestaan die dezelfde naam hebben.
  • Opgeslagen voorkeuren
    Een aantal voorkeuren worden opgeslagen voor hergebruik wanneer programma opnieuw wordt gestart.
